[JavaFX 관련 환경 설정]

0. JavaFX SDK, SceneBuilder 설치.
JavaFX SDK : 아래 링크에서 21.07 ver SDK를 다운로드, 압축 풀기
https://gluonhq.com/products/javafx/


1. 라이브러리 설정
[File-Project Structure-Libraries]에서 +버튼을 눌러 압축을 해제한 JavaFX SDK의 lib폴더를 지정

2. Main 클래스 지정
[Run - Edit Configurations - Main class] 에서 Main 클래스를 지정

3. VM option 추가
[Modify options - Add VM options] 를 추가, 다음 명령어 입력
--module-path "[JavaFX lib 위치]" --add-modules=javafx.fxml,javafx.controls


(참고)
File
└── Project Structure (Ctrl+Alt+Shift+S)
     ├── Project
     ├── SDKs
     ├── Libraries  ← (여기서 JavaFX 추가)
     └── Modules    ← (여기서 추가된 라이브러리 연결)
